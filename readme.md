# Проект новостной страницы

Разработан в качестве практической работы на курсе phpdev skillfactory.

Добавлено пять новостей.

Все заголовки и новости вымышленные и ничего общего с реальностью не имеют, все совпадению считать случайными.

Изображения загружены с сайта [unsplash.com](https://unsplash.com)

Согласно заданию, у каждой второй новости изменен фон блока на серый, а у первых двух новостей добавлен ярлык "New".
